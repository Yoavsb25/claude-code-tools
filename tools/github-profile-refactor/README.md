# github-profile-refactor — Elevate Your GitHub Profile README

Refactors and rewrites a GitHub profile README to improve readability, personal brand, organisation, and writing quality. Goes beyond cosmetic fixes — restructures content, sharpens the bio, and makes the profile actually represent the person.

## What it does

- Fetches the current profile README from GitHub
- Analyses: structure, tone, content gaps, visual hierarchy
- Rewrites with improved: headline, bio, project highlights, tech stack presentation, contact/links section
- Produces a diff for review before applying

## Requirements

- GitHub access (reads the profile repo `<username>/<username>`)
- GitHub MCP or CLI available

## Usage

```
refactor my GitHub profile
make my GitHub profile better
rewrite my GitHub README
```

The skill presents a rewritten version for approval before making any changes.
